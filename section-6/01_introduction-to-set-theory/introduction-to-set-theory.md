# 🔹 1. What is a Set?

👉 A **set** is:

> A collection of well-defined elements.

* **Collection** → group of things
* **Well-defined** → we can clearly decide whether something belongs or not

---

### ✔ Examples

* `{1, 2, 3}` → set of numbers
* `{a, b, c}` → set of characters
* `{red, green, blue}` → set of colors

---

### ❌ Not a set (not well-defined)

* `{beautiful flowers}`  
  👉 “beautiful” is subjective, so we can’t clearly decide

---

# 🔹 2. Representation of Sets

There are mainly **two ways**:

## ✅ 1. Roster Form (Listing Method)

👉 List all elements inside `{}`

Example:

```
A = {1, 2, 3, 4}
```

---

## ✅ 2. Set-Builder Form

👉 Define a rule (condition)

Example:

```
A = { x : x is a natural number less than 5 }
```

👉 Means:

```
A = {1, 2, 3, 4}
```

```
O = { o : o is an Oppo Smartphone and total number is 100 }
```
---

# 🔹 3. Types of Sets

---

## ✅ 1. Finite Set

👉 Has **limited elements**

Example:

```
A = {1, 2, 3, 4}
```

---

## ✅ 2. Infinite Set

👉 Has **endless elements**

Example:

```
N = {1, 2, 3, 4, ...}
```

---

## ✅ 3. Empty Set (Null Set)

👉 Has **no elements**

```
A = {}
```

Symbol:

```
∅
```

---

## ✅ 4. Singleton Set

👉 Only **one element**

```
A = {5}
```

---

# 🔹 4. Important Standard Sets

These are very commonly used.

---

## 🔸 Natural Numbers

```
N = {1, 2, 3, 4, ...}
```

👉 Counting numbers

---

## 🔸 Whole Numbers

```
W = {0, 1, 2, 3, 4, ...}
```

👉 Includes 0

---

## 🔸 Integers

```
Z = {..., -3, -2, -1, 0, 1, 2, 3, ...}
```

👉 Positive + negative + zero

---

## 🔸 Rational Numbers

```
Q = { x : x = p/q, where p and q are integers and q ≠ 0 }
```

👉 Fractions like:

```
1/2, 3/4, -5/2
```

---

## 🔸 Real Numbers

```
R = all rational + irrational numbers
```

👉 Includes:

* √2, π, etc.

---

# 🔹 5. Element of a Set

---

## ✅ Belongs to

```
2 ∈ A
```

👉 Means:
2 is in set A

---

## ❌ Does not belong

```
5 ∉ A
```

👉 Means:
5 is not in set A

---

# 🔹 6. Subset and Superset

---

## ✅ Subset

👉 A is a subset of B if all elements of A are in B

```
A ⊆ B
```

Example:

```
A = {1, 2}
B = {1, 2, 3}
```

👉 A ⊆ B

---

## ✅ Proper Subset

👉 A is a proper subset of B if:

* A ⊆ B
* A ≠ B

Symbol:

```
A ⊂ B
```

---

## ✅ Superset

👉 B contains A

```
B ⊇ A
```

---

## ✅ Proper Superset

```
B ⊃ A
```

---

# 🔹 7. Universal Set

👉 The **complete set of all elements under consideration**

---

# 🔹 8. Important Properties of Sets

---

## ✅ 1. Order Does NOT Matter

👉 Changing the order does not change the set.

```
A = {1, 2, 3}
B = {3, 2, 1}
```

👉 These are **exactly the same set**

```
A = B
```

---

## ✅ 2. Repetition Does NOT Matter

👉 Duplicate elements are ignored.

```
A = {1, 2, 2, 3, 3, 3}
```

👉 This is the same as:

```
A = {1, 2, 3}
```

# 🔹 10. Venn Diagram

👉 A **visual way** to represent sets

---

### How it works:

* Rectangle → **Universal set (all elements)**
* Circles → **Individual sets**
* Points inside → **elements**

---

### Example:

If:

```
A = {1, 2}
B = {1, 2, 3}
```

👉 Circle A is completely inside circle B  
👉 Shows A ⊆ B
