# 🔹 1. Union of Sets

## ✅ Definition

👉 The **union** of two sets means:

> All elements that are in A **or** in B (or in both)

---

## 🔹 Representation

```
A ∪ B
```

---

## 🔹 Example

```
A = {1, 2, 3}
B = {3, 4, 5}
```

```
A ∪ B = {1, 2, 3, 4, 5}
```

👉 Notice:

* `3` appears once (no repetition)
* Order doesn’t matter
* Union gives us a new set. So union is nothing but a set.

---

## 🔹 Set-Builder Form

```
A ∪ B = { x : x ∈ A or x ∈ B }
```

---

## 🔹 Venn Diagram

👉 Shaded region = everything inside A and B

---

# 🔹 2. Intersection of Sets

## ✅ Definition

👉 The **intersection** means:

> Elements common in both A and B

---

## 🔹 Representation

```
A ∩ B
```

---

## 🔹 Example

```
A = {1, 2, 3}
B = {2, 3, 4}
```

```
A ∩ B = {2, 3}
```

---

## 🔹 Set-Builder Form

```
A ∩ B = { x : x ∈ A and x ∈ B }
```

---

## 🔹 Venn Diagram

👉 Only the overlapping part is selected

---

# 🔹 3. Difference of Sets (Subtraction)

## ✅ Definition

👉 The **difference** means:

> Elements in A but NOT in B

---

## 🔹 Representation

```
A - B
```

(also written as `A \ B`)

---

## 🔹 Example

```
A = {1, 2, 3}
B = {2, 3, 4}
```

```
A - B = {1}
```

---

## 🔹 Set-Builder Form

```
A - B = { x : x ∈ A and x ∉ B }
```

---

## 🔹 Venn Diagram

👉 Only the part of A outside B is selected

---

# 🔹 4. Key Observations

---

## ✔ No duplicates

```
A ∪ B → unique elements only
```

---

## ✔ Order does not matter

```
{1, 2, 3} = {3, 2, 1}
```

---

## ✔ Direction matters in subtraction

```
A - B ≠ B - A
```

Example:

```
A - B = {1}
B - A = {4}
```
