# 🚀 Additional Type-Level Practice Resources

## 1. ⚔️ TypeHero

**Link:** [https://typehero.dev](https://typehero.dev)

An interactive TypeScript learning and challenge platform with curated tracks and difficulty-based challenges.

### Best for:

- Practicing TypeScript directly in the browser
- Following guided learning tracks
- Solving curated challenges
- Improving TypeScript through hands-on practice

### How to use it:

Start with beginner-friendly tracks, then slowly move to harder challenges. It is especially useful when you want a more guided and interactive experience than a GitHub repository.

---

## 2. 🎄 Advent of TypeScript

**Link:** [https://www.adventofts.com](https://www.adventofts.com)

A December-based TypeScript challenge event inspired by the Advent of Code format. It gives daily TypeScript challenges from December 1 to Christmas.

### Best for:

- Fun seasonal TypeScript practice
- Daily type-level problem-solving
- Building consistency
- Testing your skills in a challenge/event format

### How to use it:

Solve the daily challenges during the event, or use previous years’ challenges later for practice. The problems are fun, but they can become challenging quickly.

---

## 3. 🧠 TS Puzzle

**Link:** [https://ts-puzzle.com](https://ts-puzzle.com)

A puzzle-based platform focused on tricky TypeScript problems and language behavior.

### Best for:

- Practicing TypeScript puzzles
- Understanding tricky TypeScript concepts
- Improving problem-solving ability
- Identifying gaps in your TypeScript knowledge

### How to use it:

Use it as an extra practice platform after learning a concept. It is great for revision and for testing whether you really understand TypeScript’s behavior.

---

## 4. 📘 Type-Level TypeScript

**Link:** [https://type-level-typescript.com](https://type-level-typescript.com)

A written course-style resource for learning type-level programming in TypeScript. It includes structured chapters and challenges.

### Important note:

Out of 12 chapters, 4 chapters are free, including both the written material and the challenges. The remaining chapters are paid. These 4 free chapters are still very useful and can give you enough practice and understanding of how the TypeScript type system works.

### Best for:

- Learning type-level programming systematically
- Understanding the theory behind advanced TypeScript types
- Reading structured explanations
- Strengthening concepts before attempting difficult challenges

### How to use it:

Read the free chapters carefully and try the included challenges. Focus on understanding conditional types, recursive types, inference, tuple manipulation, and advanced type transformations.

---

# ✅ Final Recommendation

Do not jump between too many resources at once.

For now, focus mainly on **Type Challenges** mentioned in the previous chapter. Try completing the **Easy** and **Medium** challenges first.

Also, make sure to read the **4 free chapters** from **Type-Level TypeScript**. Those chapters alone can give you a very strong understanding of how the TypeScript type system actually works.

These additional resources are optional, but they are excellent for extra practice, deeper understanding, and improving your problem-solving skills over time.
